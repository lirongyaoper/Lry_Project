粉红色的回忆
`box` 对象是一个表示 HTML 中 `
` 标签的 HTMLElement，具有很多属性和方法。以下是 box 对象的一些常用属性和方法： 1. 属性： - `id`：表示元素的 ID。 - `className`：表示元素的 class 属性值。 - `classList`: 提供了一个类名列表，可用于添加、删除和切换类。 - `tagName`：元素的标签名（此处为 DIV） - `innerHTML`：表示元素的内部 HTML 内容。 - `outerHTML`：表示元素及其子级的 HTML 内容。 - `textContent`：表示元素包含的纯文本内容。 - `style`：提供了一个对象，可用于读取和修改元素的内联样式。 2. 方法： - `setAttribute(attrName, attrValue)`: 设置元素的属性。 - `getAttribute(attrName)`: 获取元素的属性值。 - `removeAttribute(attrName)`: 删除元素的属性。 - `hasAttribute(attrName)`: 检查元素是否存在指定属性。 - `appendChild(childElement)`: 向元素内添加子元素。 - `insertBefore(newElement, referenceElement)`: 在元素内的指定元素前插入新元素。 - `removeChild(childElement)`: 移除元素内的子元素。 - `replaceChild(newElement, oldElement)`: 替换元素内的子元素。 - `cloneNode(deepCopy)`: 复制一个新的元素实例，`deepCopy` 参数设置为 `true` 时会复制该元素的所有后代元素。 - `addEventListener(eventName, callback, options)`: 向元素添加事件侦听器。 - `removeEventListener(eventName, callback, options)`: 移除元素的事件侦听器。 此外，还有很多其他属性和方法可供使用。可以参考 [MDN 文档](https://developer.mozilla.org/zh-CN/docs/Web/API/HTMLElement) 了解更多详细信息。 需要注意的是，使用 `document.querySelector('div')` 方法只会返回符合选择器的第一个 `
` 元素。如果页面上有多个 `
` 标签，您可能需要使用 `document.querySelectorAll('div')` 获取所有匹配的元素，并对返回的 `NodeList` 进行遍历操作。
