就1个 `querySelector` 和 `querySelectorAll` 是两种常用的 JavaScript 方法，用于从 DOM（文档对象模型）中选择元素。它们的主要区别在于选择的方式和返回的结果。 1. `querySelector` `querySelector` 用于获取与指定 CSS 选择器匹配的文档中的第一个元素。如果有多个元素符合条件，它只会返回第一个匹配的元素。如果没有找到匹配的元素，它将返回 `null`。 语法： ```javascript element.querySelector(selector); ``` 示例： ```javascript // 获取页面中第一个 class="example" 的元素 const firstElement = document.querySelector('.example'); ``` 2. `querySelectorAll` `querySelectorAll` 用于获取与指定 CSS 选择器匹配的文档中的所有元素集合。返回结果是一个 `NodeList` 对象，即便只有一个或没有匹配的元素。你可以通过索引或 `forEach` 循环访问 `NodeList` 中的元素。 语法： ```javascript element.querySelectorAll(selector); ``` 示例： ```javascript // 获取页面中所有 class="example" 的元素 const allElements = document.querySelectorAll('.example'); // 遍历 NodeList 中的所有元素 allElements.forEach((element) => { console.log(element); }); ``` 总结： - `querySelector` 仅返回文档中与指定选择器匹配的第一个元素。 - `querySelectorAll` 返回文档中与指定选择器匹配的所有元素的 `NodeList` 对象。 - 当你只需要获取一个特定元素时，可以使用 `querySelector`。而如果要获取所有匹配的元素并对其进行遍历或操作，就需要使用 `querySelectorAll`。
